Modal

<modal name="[name]">[modal-html]</modal>

The modal is intentionally designed to work within components. It can be toggled using this.$root.$emit('toggleModal','[name]') within a component, or directly from HTML (e.g. <button @click="$emit('toggleModal','[name]')">)

Prop Function
name (required) Needed to trigger distinct modal.
Slot Function
general HTML content Represents the complete content of the modal to provide maximum flexibility.

TIP: create a method in your component to conveniently toggle a particular modal.
